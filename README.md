# 🚦 LAMPU LALU LINTAS 4 ARAH MENGGUNAKAN ARDUINO UNO

Berikut ini merupakan simulasi sederhana penggunaan Arduiono Uno dalam sistem lampu lalu lintas 4 arah. Sistem berkerja secara otomatis dan berulang tanpa henti dengan lampu hijau menyala secara bergiliran dengan urutan siklus : Utara → Timur → Selatan → Barat → ulangi.

## Fitur Utama
- Sistem berjalan terus menerus tanpa berhenti
- Hanya satu arah hijau dalam satu waktu
- Semua arah merah sebelum pergantian
- Urutan stabil dan konsisten
- Menggunakan fungsi agar kode rapi dan mudah dipahami

## Alur Kerja Sistem
- Semua lampu merah aktif sebagai kondisi awal
- Satu arah dipilih
- Lampu merah arah tersebut mati
- Lampu hijau menyala selama 5 detik
- Lampu kuning berkedip 3 kali
- Lampu merah menyala kembali
- Sistem lanjut ke arah berikutnya sesuai dengan urutan
